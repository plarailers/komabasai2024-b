# 第75回駒場祭（プランB）

開催日：2024年11月22日（金）～24日（日）

## 内容

【自動運転】
- train
  - 列車に書き込むコード
- master_controller
  - マスコンに書き込むコード
- wire_pole
  - 架線柱に書き込むコード
- point_switching
  - ポイントに書き込むコード
- ptcs
  - PC側の列車制御システム

【その他】
- experimental
  - 実験的なコードを置くところ
- rotary_encoder
  - ロータリーエンコーダの動作確認
- HiFreq_ADC
  - モータ電流ノイズを用いたIPSの実験
